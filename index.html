links

When creating links that open in a new tab with "target=_blank", for security purposes you must use rel="noopener noreferrer" too to avoid reverse tabnabbing

Many sites recommend just rel="noopener" but using noreferrer too seems to have better coverage for older browsers that don't support noopener

Bad example

a href="https://randomsite.com" target="_blank"

Good example

a href="https://randomsite.com" target="_blank" rel="noopener noreferrer"
